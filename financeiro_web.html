⬆️ Exportar Backup

Gera um arquivo .json com todos os dados do sistema: lançamentos, cadastros (contas, empresas, tipos, centros de custo) e configurações. Salve este arquivo em um local seguro como Google Drive, OneDrive ou Dropbox.

💡 Dica: Salve o backup no Google Drive ou OneDrive para ter acesso de qualquer lugar.
🔁 Recomendação: Faça backup pelo menos uma vez por semana ou sempre após lançamentos importantes.
🔒 O arquivo contém dados financeiros — mantenha-o em local seguro e de acesso restrito.

⬆️ Importar Backup

Restaura os dados a partir de um arquivo de backup .json gerado por este sistema. Você pode escolher entre substituir todos os dados ou mesclar com os dados existentes (apenas novos lançamentos serão adicionados).
